# CwTest

## How To

### Открыть документ по пути http://localhost:4200/document/view
Добавить аннотацию можно с помощью правого клика на странице документа

Удалить аннотацию можно с помощью двойного клика левой кнопкой мыши по аннотации

## Недостатки
1. Прослойка для отрисовки аннотаций получилась перегруженной. Каждый раз при перемещении аннотации приходится обновлять весь объект, чтобы обновить позицию аннотации. То же самое со скейлингом. Скейлинг вызывает пересчёт позиции, что приводит к обновлению объекта.
2. `DocumentRendererService` получился немного "раздут". Можно логику для управления аннотациями вынести в отдельный сервис, оставить сервису логику только по отрисовки.
3. При перетаскивании аннотации хотелось бы автоматическую прокрутку, если выходим за видимую часть страницы

## Плюсы
1. Код для просмотра документа вынесен в отдельный модуль, можно переиспользовать
2. Логика по отрисовки документа вынесена в отдельный сервис
3. В компонентах минимум логики - только обработка событий, вызов методов сервиса
4. Перетаскивание аннотацией ограничено страницей документа, невозможно вынести аннотацию из страницы
5. Код полностью типизирован
6. Использованы сигналы
