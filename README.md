# CwTest

## How To

### Добавить аннотацию можно с помощью правого клика на странице документа
### Удалить аннотацию можно с помощью двойного клика левой кнопкой мыши на странице документа

## Недостатки
### 1. Прослойка для отрисовки аннотаций получилась перегруженной. Каждый раз при перемещии аннотации приходится обновлять весь объект, чтобы обновить позицию аннотации. То же самое со скейлингом. Скейлинг вызывает пересчёт позиции, что приводит к обновлению объекта.
### 2. `DocumentRendererService` получился немного "раздут". Можно логику для управления аннтоациями вынести в отдельный сервис, оставить сервису только только по отрисовке.


## Плюсы
### 1. Код для просмотра документа вынесен в отдельный модуль, можно использовать обособленно от всего приложения 
### 2. Логика по отрисовке документа вынесена в отдельный сервис
### 3. В компонентах минимум логику - только обработка событий, вызов методов сервиса
### 4. Перетаскивание аннтоаций ограничено страницей, невозможно вынести аннотацию из страницы
### 5. Код полностью типизирован
### 6. Использованы сигналы
